PROJECT DESCRIPTION

The purpose of this site is to showcase my abilities as a Python developer and what Flask can do as a framework for web applications.

The site works on computers and mobile devices, and implements security mechanisms to prevent Cross Site Scripting (XSS) attacks, SQL injections, and other threats.

The backend runs on nginx on a Linode server.

In the process of finishing this project, I got more comfortable working with both frontend and backend web development, including:

  • Python 3
  • SQL / MySQL / SQLAlchemy / SQLite3
  • Flask
  • HTML5
  • CSS3
  • Bootstrap
  • WTForms
  • Bcrypt
  • nginx
  • SSH
  • HTTPS certificates
  • Linux bash commands + cloud server administration
  • Web Security
  • virtualenv
  • pip
